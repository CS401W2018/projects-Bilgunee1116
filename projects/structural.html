Structural Tags

1. What goes in the header?

Contains the website title, logo, and possibly navigation links or a search bar.

2. What goes in the Nav? Can the Nav go in the header? Does it have to go in the header? Does the Nav contain links to another website?

Holds links for site navigation, like "Home" or "About." It can go in the header but does not have to. It can also link to external websites.

3. What goes in Main?

Where the main content is displayed, like text, images, or videos.

4. What goes in the Footer?

Includes additional info like copyright, privacy policy, and contact links.

5. HTML Validation?

Checks if the code is correct and follows web standards to ensure proper display on all devices.
