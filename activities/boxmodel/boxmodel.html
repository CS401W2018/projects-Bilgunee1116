Box Model

1. Block tags take up the full width and start on a new line, while inline tags only take up as much width as needed and stay in line.
2. Margin is the space outside an element, border is the outline around it, and padding is the space between content and the border.
3. CSS units include absolute units and relative units
4. Use the figure tag to group media with captions, improving semantics and accessibility.
