1. Internet

The Internet is like a big network that connects computers and phones around the world. It helps people share information, watch videos, and stay in touch with others.

2. Web

The Web is a part of the Internet where you can visit websites and find information.

3. ASCII (American Standard Code for Information Interchange

ASCII is a system that turns letters, numbers, and symbols into codes for computers to understand.

4. Unicode

Unicode is a system that helps computers show text in all languages, such as English, Spanish, or Mongolian. It includes many more characters than ASCII.

5. UTF-8

UTF-8 is a way to store text using Unicoide that works with many languages. It saves space and works with older systems.

6. URL (Uniform Resource Locator)

URL is the address you type to visit website. For example, it could be wwww.facebook.com. It shows where a webpage or file is on the Web.

7. Domain Name

A domain name is the name of a website that's easy to remember, like youtube.com. It helps people visit a site without using code.

8. HTTP (HyperText Transfer Protocols)

HTTP is the system lets the browser get and show webpages. It helps people's device and website servers share information.

9. HTTPS (HyperText Transfer Protocol Secure)

HTTPS can protects our information when we visit websites. It keeps our data secure, especially for online shopping.

10. IP Address (Internet Protocol Address)

IP address is a special number that identifies a device, such as a computers or phone, on a network.

11. DNS (Domain Name System)

DNS is a system that changes website names such as facebook.com, into numbers (IP address), into numbers (IP addresses) that computers understand.

12.W W3C Recommendations

W3C Recommendations are helpful rules that make websites work smoothly on all devices and browsers.

13. XML (eXtensible Markup Language)

XML is a way to organize and share information in a clear, easy to read format.

14. XHTML ( eXtensible HyperText Markup Language)

XHTML can make websites cleaner and easier for computers to process.

15. HTML

HTML is the basic language used to make websites.

16. HMTL 5

HTML5 is the newest version of HTML that added features such as videos and animations. It helps website work better on phones.
