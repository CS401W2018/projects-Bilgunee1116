CSS

1. Incline CSS is written inside an element, embedded CSS is placed in the HTML file's style tag, and external CSS is stored ina separate file and linked to the HTML.
2. External CSS is best because it keeps styles consistent, improves performance, and makes updates easier while keeping HTML clean.
3. A selector targets elements baseed on type, class, ID, or attributes to apply styles efficiently.
